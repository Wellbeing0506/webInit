# webInit
website Project initial item

# Environment
1. NodeJS v4.2.0
2. NPM 2.14.7
3. express 4.13.4

### Proecss 
1. setup HTTPS server 
2. User login with account/password (passport)
3. cookie and session check login
4. DB switch
  	
### Execute 
>$cd webinit

>$npm install 

>$node bin/www or $npm start

or

>$pm2 start bin/www --name "website:Port"
